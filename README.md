# Módulos 24, 25 e 26: SQL
Projeto desenvolvido durante o curso Profissão: Cientista de Dados. O objetivo do projeto foi a aplicação de funções básicas e avançadas de SQL. 

## Módulo 24: Banco de Dados + SQL
  ### Etapas
  - Etapa 1: Consultar todas as linhas e colunas do banco de dados de vendas de e-commerce (SELECT * FROM);
  - Etapa 2: Aplicação da função LIMIT para delimitar o tamanho da resposta;
  - Etapa 3: Varificação da média do valor por unidade e da soma das unidades vendidas, com AVG() e SUM(), que criaram colunas que foram renomeadas com AS;
  - Etapa 4: Consulta da coluna TOTAL_GASTO com CAST() para aplicar uma multiplicação às colunas de preço por unidade e quantidade comprada;
  - Etapa 5: Verificação da média de TOTAL_GASTO com AVG(CAST());
  - Etapa 6: Resolução de questões teóricas sobre SQL.

## Módulo 25: Operações Básicas com SQL
  ### Etapas
  - Etapa 1: Consulta de produtos distintos com SELECT DISTINCT;
  - Etapa 2: Consulta para verificar os clientes únicos com COUNT();
  - Etapa 3: Consulta para retornar uma coluna com valores distintos e preço por unidade em que o valor era maior que 50. Para isso, utilizaram-se os comandos DISTINCT, CAST() e WHERE;
  - Etapa 4: Consulta para retornar o ID das 5 compras com maior gasto e o valor gasto em cada uma. Para isso, foram utilizados os comandos ORDER BY, CAST(), DESC e LIMIT;
  - Etapa 5: Consulta para retornar os produtos e média do preço por unidade de cada produto, ordenando do maior para o menor. Aqui, foram utilizados os comandos AVG(), CAST(), GROUP BY, ORDER BY e DESC;
  - Etapa 6: Consulta para retornar o ID dos 3 clientes que mais realizaram compras e o total de compras realizadas. Aqui, foram utilizados todos os comandos COUNT(), GROUP BY, ORDER BY, DESC e LIMIT.

## Módulo 26: Operações Avançadas com SQL
  ### Etapas
  - Etapa 1: Aplicação de INNER JOIN para evitar valores vazios no conjunto de dados final;
  - Etapa 2: Elaboração do dashboard interativo com os dados do e-commerce.

## Resultados 
Neste projeto, foi possível compreender amplamente a aplicação da linguagem SQL, do básico ao avançado, e implementa-la a um conjunto de dados reais. Por fim, foi possível explorar as possibilidades da criação de dashboards com Power BI.
